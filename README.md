# Design and Analysis of High-Efficiency Truss Bridge

### Overview

This project involves the design and analysis of a 30 m span steel truss bridge using STAAD.Pro. The objective was to achieve a high Load Carrying Capacity to Self-Weight (LCC/SW) ratio while ensuring structural safety and efficiency.

### Problem Statement

A truss bridge was designed to safely carry multiple truck loads under realistic loading conditions. The structure was analyzed to ensure all members remain within permissible stress and deflection limits.

### Methodology

1. Modeled a 2D steel truss bridge in STAAD.Pro

2. Defined geometry, supports, and boundary conditions

3. Applied dead load and moving truck loads

4. Performed structural analysis to evaluate:
- Axial forces (tension/compression)
- Member behavior
- Deflections

### Key Features

1. Efficient truss configuration selection

2. Consideration of real-world loading conditions

3. Structural optimization for weight and cost

4. Safety checks for all members

### Results

1. Structure remained stable under applied loads

2. All members satisfied strength and serviceability criteria

3. Achieved an efficient and economical truss design
